# NETFLIX (CLONE)

## Como ver este projeto:

### Criar um diretório para o projeto e clone repositório
````
mkdir projet
cd project
git clone https://github.com/RaphaelLima123/netflixclone
````

### Ir para o diretório do projeto e instalar dependências(NPM OU YARN):
````
cd netflixclone
npm install
yarn start
````
